# Accessibility Rules

## Required Coverage

- Screen reader labels
- Touch target size
- Focus order
- Reduced motion
- Color contrast
- Error messaging

## Rules

- Interactive controls need accessible names.
- Icon-only buttons need labels or tooltips.
- Error messages must be associated with the field or action they describe.
- Do not rely on color alone to convey state.
- Motion-heavy interactions must respect reduced motion settings when applicable.
- Critical flows require manual accessibility review.
